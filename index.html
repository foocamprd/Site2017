Acerca del Foocamp

¿Qué es?

FooCamp es una plataforma a través de la cual estudiantes, profesionales, académicos y entusiastas del Software comparten conocimiento y al mismo tiempo establecen nuevos lazos profesionales y de amistad.

¿Quién puede participar?

Profesionales, estudiantes, académicos, entusiastas y público en general. ¡Todo el que quiera compartir y/o expandir su propio conocimiento es bienvenido!

Sesiones

An Irresponsably Basic Introduction to Neural Networks

"Introducción Irresponsablemente Básica a Neural Networks"

Sala I9:00AM

El internet está de cabeza hablando del impacto de AI, Machine Learning y Deep Learning. La gente hablando de ANN, RNN, GAN, CNN, LSTM (Y quien sabe que otro acrónimo inventaron la semana pasada). En esta charla/lab, vamos a explicar en términos bien simples que es una Neural Network, cuáles son sus componentes, y como se implementa una (sin ningún framework).

Por: Carlos Carbonell López.

Seguridad en SaaS

"Software Security testing when using SaaS."

Sala II9:00AM

Veremos una demo en Vivo del poder de esta integración, Workflows y Formularios electronicos agiles.

Por: Julissa Mateo Abad.

Aplicaciones inteligentes

"Xamarin y Microsoft Cognitive Services"

Sala II 11:00AM

Desarrollar una aplicación con servicios de inteligencia artificial suena una tarea complicada, pero hoy en día existen APIs y SDKs que simplifican esta integración, pues encapsulan toda la inteligencia artificial que hay detrás de un servicio inteligente y la exponen a través de una llamada a un servicio web para obtener un resultado (por ejemplo, reconocimiento de voz, detección de rostros en una imagen, traducción en tiempo real), permitiendo al desarrollador enfocarse en la implementación y la creación de una solución informática poderosa, segura y confiable.

Por: Yhorby Matias.

Kit de supervivencia para Java 8

"Como prepararse para Java 9"

Sala II3:00PM

Java 8 lanzado en marzo del 2014 (hace tres años),es quizás el cambio más significativo en la historia del lenguaje, dado que incorpora varias novedades siguiendo la tendencia de otros lenguajes como la incorporación de lambdas,la API Stream y una demandada nueva API para el manejo de fechas entre otras muchas características más.

La próxima versión está a la vuelta de la esquina con fecha de lanzamiento de julio de 2017. La característica más esperada es modularidad en Java 9, sin embargo hay aún más a esta nueva versión.

En esta charla abordaremos en forma de repaso las principales funcionalidades de Java 8 y las nuevas funcionalidades de Java 9.

Por: Eudris Cabrera.

Django Reloaded

Sala II3:00PM

La productividad es uno de los aspectos más importante en el desarrollo de aplicaciones web y en el desarrollo de software en general. Es usual que cada proyecto venga acompañado de una fecha de entrega muy ajustada, lo que hace imprescindible disponer de toda la ayuda posible para lograr los objetivos.

En esta desconferencia veremos las características más destacadas del framework web Python más usado: Django.

Veremos como Django nos ayuda a ser más productivos, así como un compendio de mejores prácticas y apps de terceros para turbo-cargarlo y hacer del desarrollo web toda una delicia.

Por: Emilio Ferreyra.
